# 📚 Library Management System

A console-based Library Management System built in C++ that enables efficient management of books, members, and borrowing records. The project demonstrates practical implementation of Data Structures and File Handling concepts.

⭐ If you found this project useful, consider giving it a star on GitHub!

## 📖 Project Overview

The Library Management System allows administrators to manage books and members, while users can borrow, return, and search for books. All records are stored in text files, so data remains available even after the program is closed.

## ✨ Features

* 🔐 User registration and login system
* 👨‍💼 Admin dashboard
* 📖 Add, view, and manage books
* 👥 Add and manage members
* 📚 Issue and return books
* ⭐ Premium and normal membership support
* 🔍 Search books by ID, title, or author
* 📅 Book issue and return tracking
* 📊 Priority queue–based book sorting
* 💾 File-based data persistence

## 🛠️ Technologies Used

* C++
* Doubly Linked Lists
* Priority Queue
* File Handling
* Dynamic Memory Management

## ⚙️ Requirements

Before running the project, make sure you have:

* A C++ compiler (GCC, MinGW, MSVC, or Clang)
* C++11 or later
* Command Prompt, Terminal, or Visual Studio

## ▶️ How to Run

### Windows (MinGW)

```bash
g++ main.cpp -o library.exe
.\library.exe
```

### Linux / macOS

```bash
g++ main.cpp -o library
./library
```

### Visual Studio

1. Open the project in Visual Studio.
2. Build the solution (**Ctrl + Shift + B**).
3. Run the program (**Ctrl + F5**).
